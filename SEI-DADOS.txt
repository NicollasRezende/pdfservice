# Estruturas de Dados e Serviços SEI - Detalhamento Completo

## 1. ESTRUTURAS DE DADOS SEI

### **Andamento**
| Campo | Descrição |
|-------|-----------|
| IdAndamento | Identificador interno do andamento |
| IdTarefa | Identificador da tarefa associada |
| IdTarefaModulo | Identificador da tarefa de módulo |
| Descricao | Texto do andamento |
| DataHora | Data/hora em que o andamento foi gerado |
| Unidade | Dados da unidade que gerou o andamento (ver estrutura Unidade) |
| Usuario | Dados do usuário que gerou o andamento (ver estrutura Usuario) |
| Atributos | Conjunto de atributos do andamento (ver estrutura AtributoAndamento) |

### **AndamentoMarcador**
| Campo | Descrição |
|-------|-----------|
| IdAndamentoMarcador | Identificador interno do andamento de marcador |
| Texto | Texto associado com o andamento |
| DataHora | Data/hora em que o andamento foi gerado |
| Usuario | Dados do usuário que gerou o andamento (ver estrutura Usuario) |
| Marcador | Dados do marcador utilizado (ver estrutura Marcador) |

### **Anotacao**
| Campo | Descrição |
|-------|-----------|
| ProtocoloProcedimento | Número do processo visível para o usuário, ex: 12.1.000000077-4 |
| Descricao | Texto da anotação |
| SinPrioridade | S/N – Sinalizador indicando se a anotação é prioritária |

### **ArquivoExtensao**
| Campo | Descrição |
|-------|-----------|
| IdArquivoExtensao | Identificador interno do SEI relativo a extensão |
| Extensao | Texto da extensão (ex.: pdf, ods, doc, ppt,...) |
| Descricao | Descrição da extensão |

### **Assinatura**
| Campo | Descrição |
|-------|-----------|
| Nome | Nome do assinante |
| CargoFuncao | Cargo ou função utilizado no momento da assinatura |
| DataHora | Data/hora em que ocorreu a assinatura |
| IdUsuario | ID do usuário |
| IdOrigem | ID origem associado com o usuário no SIP |
| IdOrgao | ID do órgão do usuário |
| Sigla | Sigla do usuário |

### **Assunto**
| Campo | Descrição |
|-------|-----------|
| CodigoEstruturado | Código do Assunto, ex.: 00.01.01.01 |
| Descricao | Descrição do assunto |

### **AtributoAndamento**
| Campo | Descrição |
|-------|-----------|
| Nome | Nome do atributo |
| Valor | Valor do atributo |
| IdOrigem | Identificador auxiliar associado com este atributo |

### **AtributoOuvidoria**
| Campo | Descrição |
|-------|-----------|
| Id | Identificador do atributo da ouvidoria, ex.: P |
| Nome | Nome interno do atributo, ex.: RELACAO |
| Titulo | Título do atributo para exibição no formulário |
| Valor | Valor do atributo para exibição |

### **Bloco**
| Campo | Descrição |
|-------|-----------|
| IdBloco | Número do bloco |
| Unidade | Dados da unidade que gerou o bloco (ver estrutura Unidade) |
| Usuario | Dados do usuário que gerou o bloco (ver estrutura Usuário) |
| Descrição | Descrição do bloco |
| Tipo | Tipo do bloco (A=Assinatura, R=Reunião ou I=Interno) |
| Estado | Estado do bloco (A=Aberto, D=Disponibilizado, B=Recebido, R=Retornado ou C=Concluído) |
| SinPrioridade | S/N – Indica se o bloco foi sinalizado como prioritário |
| SinRevisao | S/N – Indica se o bloco foi sinalizado como revisado |
| UsuarioAtribuicao | Dados do usuário para o qual o bloco foi atribuído |
| UnidadesDisponibilizacao | Dados das unidades configuradas para disponibilização |

## 3. SERVIÇOS SIP - ENTRADA/SAÍDA

### **Usuario (Estrutura SIP)**
| Campo | Descrição |
|-------|-----------|
| StaOperacao | C–Cadastrar, A–Atualizar, E–Excluir, D–Desativar, R–Reativar |
| IdOrigem | ID associado com o usuário no sistema de origem |
| IdOrgao | ID do órgão do usuário |
| Sigla | Sigla do usuário |
| Nome | Nome do usuário (Registro Civil) |
| NomeSocial | Nome do usuário (Social) |
| Cpf | CPF do usuário |
| Email | Email Institucional do usuário |

### **Permissao (Estrutura SIP)**
| Campo | Descrição |
|-------|-----------|
| StaOperacao | A–Atualizar, E–Excluir |
| IdSistema | ID do sistema no SIP |
| IdOrgaoUsuario | ID do órgão do usuário no SIP |
| IdUsuario | ID do usuário no SIP |
| IdOrigemUsuario | ID do usuário no sistema origem |
| IdOrgaoUnidade | ID do órgão da unidade no SIP |
| IdUnidade | ID da unidade no SIP |
| IdOrigemUnidade | ID da unidade no sistema origem |
| IdPerfil | ID do perfil no SIP |
| DataInicial | Data de início da permissão |
| DataFinal | Data de término da permissão |
| SinSubunidades | S/N - estende às subunidades |

### **listarPermissao (SIP)**
